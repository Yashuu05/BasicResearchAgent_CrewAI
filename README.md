# **Basic Research Agent using CrewAI**
---

### ABOUT
- This is my first AI Agent project which leverages the **CrewAI** framework to build and deploy the basic research agent.
- The Agent uses **Ollama** as LLM provider and **LLama 3.1: 8b** as LLM.
- The purpose of AI agent is to use existing **tools** to search the web, extract the information and provide a simple research content.
- The script uses the similar logic as provided in CrewAI documentation : [See Documentation](https://docs.crewai.com/)
